# UI SVG Assets

Place SVG files here as `{name}.svg`.

Load in JS: `uiSvg('name', size)` returns `<img src="Assets/ui/name.svg" class="ui-icon">`.
CSS class `.ui-icon` handles sizing and pixelated rendering.

Naming convention: lowercase-kebab-case matching the UI element.
Examples: `play-btn.svg`, `discard-btn.svg`, `bag.svg`, `gold-coin.svg`
